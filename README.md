### 🛠️ The "Prototyping PM" Philosophy
I believe that the best technical product managers don't just write static specs—they build functional prototypes. I leverage code to validate complex workflows, stress-test third-party APIs, and gather early user feedback. This hands-on approach accelerates alignment with engineering, eliminates requirement ambiguity, and reduces overall launch risk.
